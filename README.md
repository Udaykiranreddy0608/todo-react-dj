# todo-react-dj
A Simple TODO app buit using ReactJS and Django

### Heroku 

#### Deployment 

```
$ heroku create todo-react-dj
$ heroku git:remote -a todo-react-dj
$ heroku buildpacks:add --index 1 heroku/nodejs
```
#### Heroku cli scale and status 

```
$ heroku ps 
$ heroku ps:scale web=0
```

### React App

#### Build 

```
yarn build
```

#### Start React project 

```
yarn start 
```
